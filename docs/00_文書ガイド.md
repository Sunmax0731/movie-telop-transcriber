# 文書ガイド

## 1. 目的
この文書は、`movie-telop-transcriber` の文書群を、利用者向け、開発者向け、工程・検証向けに読み分けられるようにするための入口である。

## 2. 最初に読む文書

### 2.1 利用者向け
- 導入したい:
  [12_導入手順書.md](12_導入手順書.md)
- 使い方を知りたい:
  [16_利用ガイド.md](16_利用ガイド.md)
- 配布物の内容を知りたい:
  [11_配布構成と同梱物.md](11_配布構成と同梱物.md)
- 既知制約を知りたい:
  [08_既知不具合と制約一覧.md](08_既知不具合と制約一覧.md)

### 2.2 開発者向け
- 現在の工程と issue 状態を知りたい:
  [02_開発工程.md](02_開発工程.md)
- すぐ再開したい:
  [17_現在状態サマリ.md](17_現在状態サマリ.md)
- 全体仕様を把握したい:
  [03_仕様書.md](03_仕様書.md)
- 設計を把握したい:
  [04_基本設計書.md](04_基本設計書.md)、[05_詳細設計書.md](05_詳細設計書.md)
- テスト観点を知りたい:
  [06_テスト計画書.md](06_テスト計画書.md)
- 出力と評価仕様を知りたい:
  [spec/04_出力仕様.md](spec/04_出力仕様.md)、[spec/06_QCDS評価仕様.md](spec/06_QCDS評価仕様.md)、[spec/08_MainPageViewModel責務分割方針.md](spec/08_MainPageViewModel責務分割方針.md)

### 2.3 工程・検証向け
- 最新 release の要点を知りたい:
  [13_リリースノート.md](13_リリースノート.md)

  [18_配布物manifest.md](18_配布物manifest.md)

## 3. 文書の読み分け

| 区分 | 主な対象読者 | 主な文書 |
| --- | --- | --- |
| 利用者向け | 初めて使う利用者、導入担当者 | `README.md`、`12_導入手順書.md`、`16_利用ガイド.md`、`11_配布構成と同梱物.md` |
| 開発者向け | 実装担当者、保守担当者 | `02_開発工程.md`、`03_仕様書.md`、`04_基本設計書.md`、`05_詳細設計書.md`、`06_テスト計画書.md` |
| 工程・検証向け | Issue 管理者、release 担当者、評価担当者 | `08_既知不具合と制約一覧.md`、`13_リリースノート.md`、`17_現在状態サマリ.md`、`18_配布物manifest.md`、`docs/test-results/`、`docs/spec/` |

## 4. 更新ルール
- 利用者向け導線を変えたときは、`README.md`、`12_導入手順書.md`、`16_利用ガイド.md` を見直す。
- 工程や active Issue が変わったときは、`02_開発工程.md` を更新する。
- release 判定の根拠が増えたときは、`13_リリースノート.md` と `docs/test-results/` の参照先を見直す。
- 文書を追加したときは、この文書から辿れるようにする。
