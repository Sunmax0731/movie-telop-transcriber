## 2. v1.1.0 リリース概要
`v1.1.0` は、正式版 `v1.0.0` のあとに優先改善として扱った 4 つの作業を反映した release である。

主な追加内容:
- `qcds_suite` 3 本の actual OCR canonical を追加
- installer 後の OCR readiness 診断を標準化
- `MainPageAnalysisOutputCoordinator` による解析出力 orchestration 分割
- `Test-ReleaseSmoke.ps1` による release 前 smoke / 回帰確認の標準化

## 3. v1.1.0 配布物

```text
movie-telop-transcriber-win-x64-v1.1.0.zip

```

## 4. v1.1.0 導入前提
- 対象 OS は Windows x64 とする
- `.NET 10 Desktop Runtime x64` を前提にする
- 実動画 OCR を使う場合は Python `3.10.x`、PaddlePaddle `3.2.0`、PaddleOCR `3.5.0`、`PP-OCRv5_server_det` / `PP-OCRv5_server_rec` を installer または手動導入で準備する
- 詳細は [docs/12_導入手順書.md](12_導入手順書.md) を参照する

## 5. v1.1.0 の既知制約
- `self-contained publish` は 2026-04-30 時点で `0xC000027B` / `0xE0434352` による起動失敗が残っている
- OCR runtime は zip に含めず、installer または手動導入で取得する
- 実動画系 canonical QCDS はまだ拡張余地がある
