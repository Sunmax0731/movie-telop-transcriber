## 2. v1.2.0 リリース概要
`v1.2.0` は、`v1.1.0` のあとに加点改善として扱った 4 つの作業を反映した release である。

主な追加内容:
- package zip に `START_HERE.md` を追加
- 利用者向け / release 担当向けの短縮ガイドを追加
- OCR runtime pack を標準 asset にしない判断を文書化
- repo 外の rights-cleared 実動画でも使える QCDS canonical workflow を追加

## 3. v1.2.0 配布物
GitHub Release `v1.2.0` では、次の asset を公開対象とする。

```text
movie-telop-transcriber-win-x64-v1.2.0.zip
movie-telop-transcriber-win-x64-v1.2.0.zip.sha256
Install-MovieTelopTranscriber.ps1
Install-MovieTelopTranscriber.ps1.sha256
```

## 4. v1.2.0 導入前提
- 対象 OS は Windows x64 とする
- `.NET 10 Desktop Runtime x64` を前提にする
- 実動画 OCR を使う場合は Python `3.10.x`、PaddlePaddle `3.2.0`、PaddleOCR `3.5.0`、`PP-OCRv5_server_det` / `PP-OCRv5_server_rec` を installer または手動導入で準備する
- 詳細は [docs/12_導入手順書.md](12_導入手順書.md) を参照する

## 5. v1.2.0 の既知制約
- `self-contained publish` は 2026-04-30 時点で `0xC000027B` / `0xE0434352` による起動失敗が残っている
- OCR runtime は zip に含めず、installer または手動導入で取得する
- 実動画系 canonical workflow は追加済みだが、rights-cleared 実動画の実測レポート本数は今後も増やす余地がある
