## 1. 文書情報
- 対象 release:
  `v1.2.1`
- 位置づけ:
  Store 配布の Python 3.12 / 3.13 を含む Python 3.10-3.13 対応の patch release
- 関連文書:
  [00_文書ガイド.md](00_文書ガイド.md)、[18_配布物manifest.md](18_配布物manifest.md)、[11_配布構成と同梱物.md](11_配布構成と同梱物.md)

## 2. v1.2.1 の概要
`v1.2.1` は installer と OCR runtime 構築フローの互換性を補強する patch release である。主目的は、Microsoft Store などで入る Python 3.12 / 3.13 を含む Python 3.10-3.13 (64-bit) で導入を完走できるようにすることにある。

主な変更点:
- installer の Python 検出を `py -3.13`、`py -3.12`、`py -3.11`、`py -3.10`、`python`、`python3` の順に拡張
- 実際に解決された interpreter の version と 64-bit 判定を導入前に確認
- OCR runtime の依存を `paddlepaddle==3.2.2`、`paddleocr==3.5.0` に更新
- 導入手順、既知制約、manifest、README を Python 3.10-3.13 前提へ更新

## 3. v1.2.1 の配布物
GitHub Release `v1.2.1` では、次の asset を公開対象とする。

```text
movie-telop-transcriber-win-x64-v1.2.1.zip
movie-telop-transcriber-win-x64-v1.2.1.zip.sha256
Install-MovieTelopTranscriber.ps1
Install-MovieTelopTranscriber.ps1.sha256
```

## 4. 導入前提
- 対象 OS は Windows x64
- `.NET 10 Desktop Runtime x64` を利用者環境に用意する
- 実動画 OCR を使う場合は Python `3.10.x` / `3.11.x` / `3.12.x` / `3.13.x` の 64-bit 環境を使う
- OCR runtime の依存は `paddlepaddle==3.2.2`、`paddleocr==3.5.0`
- OCR モデルは `PP-OCRv5_server_det` / `PP-OCRv5_server_rec`

詳細な導入手順は [12_導入手順書.md](12_導入手順書.md) を参照する。

## 5. 既知制約
- `self-contained publish` は引き続き標準配布経路ではない
- Python runtime、PaddlePaddle / PaddleOCR の Python package、PaddleOCR モデル本体は zip へ同梱しない
- Python 未導入 PC で installer 単体完結にはならず、対象 Python の事前導入または別配布の runtime pack が必要

## 6. 次の改善候補
- Python 未導入 PC 向けの runtime 同梱方式の検討
- self-contained publish 問題の継続調査
- rights-cleared 実動画を使った canonical 検証の継続拡充
