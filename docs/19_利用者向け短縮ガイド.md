# 利用者向け短縮ガイド

## 1. これだけやれば使い始められる
1. GitHub Release から zip を取得して展開する
2. zip 直下の `START_HERE.md` を開く
3. `Install-MovieTelopTranscriber.cmd` を実行する
4. 導入後に `MovieTelopTranscriber\app\MovieTelopTranscriber.App.exe` を起動する
5. サンプル動画で 1 回 `解析` を実行する

## 2. 困ったときに見る文書
- 導入で止まった:
  [12_導入手順書.md](12_導入手順書.md)
- OCR readiness や既知制約を確認したい:
  [08_既知不具合と制約一覧.md](08_既知不具合と制約一覧.md)
- 普段の操作手順を確認したい:
  [16_利用ガイド.md](16_利用ガイド.md)

## 3. 最低限の確認項目
- `Test-MovieTelopTranscriberOcrReadiness.ps1` で `ready` になる
- 短い動画で `segments.json` と `segments.srt` が出力される
- 必要なら `segments.csv` と `frames.csv` を Excel で開ける
