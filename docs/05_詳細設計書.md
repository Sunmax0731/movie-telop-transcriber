# #85 プレビュー詳細設計
- `TimelineSegment` と `ResultRow` は、表示文字列に加えて `FrameIndex`、`TimestampMs`、`SegmentId`、必要に応じて `DetectionId` を持つ。
- `MainPageViewModel` は `PreviewImagePath`、`PreviewImageWidth`、`PreviewImageHeight`、`PreviewDetections` を公開する。
- `PreviewDetections` は `PreviewDetectionOverlay` として、OCR detection ID、認識文字、信頼度、bounding box、強調状態を保持する。
- segment 選択時は、segment の開始・終了時刻内で認識文字が一致するフレームを優先し、なければ detection を持つ近傍フレームを使う。
- overlay の座標変換は UI 側で実施し、OCR worker 側で元フレーム座標へ逆変換済みの値を前提にする。
- 画像ロードは `BitmapImage` に任せ、overlay は画像表示領域の `SizeChanged`、画像ロード完了、検出コレクション変更時に再描画する。
