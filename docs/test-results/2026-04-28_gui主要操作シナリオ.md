## 3. 検出事項
### 3.1 Status セクションがはみ出す
- 内容: メイン画面左ペインの Status セクションが縦方向にはみ出し、全内容を確認できない。
- 対応: 左ペイン全体を縦スクロール可能に修正。
- 再確認: 2026-04-28 にスクロールできることを確認済み。

### 3.2 サンプル用 sidecar スクリプトの誤用余地
- 内容: 任意動画の `frames/` に `materialize_sidecars.py` を実行すると、サンプル動画用 OCR 結果が書き込まれる。
- 判断: スクリプトは `sample_basic_telop.mp4` 専用の検証補助であり、実 OCR ではない。
- 対応: `test-data/basic_telop/README.md` に注意書きを追記。

### 3.3 文字サイズとレイアウト密度
- 内容: スクロールは可能になったが、Status を含む一部の文字サイズが大きく、画面全体のバランスが悪い。
- 判断: 左右ペインは補助情報、中央ペインはプレビューとタイムライン確認を主目的とするため、状態カードの値に大きな見出しスタイルを使わない構成へ寄せる。
- 対応: 共通タイポグラフィスタイルを追加し、メイン画面、設定画面、出力画面の見出し、項目名、値、補足文の階層を調整。
- 再確認: 必要。

### 3.4 左右ペイン幅とパス表示
- 内容: 左右のバランスを取るため、左ペインと右ペインの幅を統一する必要がある。
- 対応: 右ペインに合わせ、左ペインと右ペインを 340px に統一。
- 内容: Status の Export、Log、Work は項目名とパスを横並びにすると読みづらい。
- 対応: Status カードの項目名、値、補足文を縦並びに変更し、パスを改行表示。
- 再確認: 必要。

### 3.5 言語設定
- 内容: 言語設定が必要。
- 対応: 言語選択を左ペインと設定画面に追加。初期値は OS の UI 言語から日本語、英語、中国語、韓国語のいずれかを選択し、未対応言語の場合は英語にする。
- 対応: 選択言語に応じて、メイン画面、設定画面、出力画面の主要な固定ラベル、ボタン、設定サマリ、Status 項目名を更新する。
- 再確認: 必要。

### 3.6 項目説明とボタン表示
- 内容: 項目説明が項目名の下に常時表示され、情報量が多い。
- 対応: 設定サマリ、Status、入力欄の説明をツールチップへ移し、画面上は項目名と値を優先表示する。
- 内容: 再実行ボタンの文字列が横3分割のボタン領域からはみ出す。
- 対応: 右ペイン下部の再実行ボタンを縦並びの全幅ボタンに変更。
- 内容: 画面内のアプリ名と概要文が冗長。
- 対応: トップ領域からアプリ名と概要文を削除し、動画パス入力と操作ボタンだけを配置。
- 再確認: 必要。
