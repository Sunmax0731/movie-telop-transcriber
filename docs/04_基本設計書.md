## 4. 設計方針
- GUI、本体制御、OCR ワーカー、属性解析、統合、出力を責務分離する
- 実行単位 `run_id` を軸に中間成果物と最終成果物を整理する
- タイムラインと一覧とプレビューの同期を設計前提に置く
- WinUI 3 採用を前提に、UI 状態管理の方式を先に定義する
